## 📋 What is This Project?

A professional web-based system that automates sending course enrollment emails to students. It handles bulk email operations safely, tracks delivery status, and provides an intuitive interface for management.

---

## 🎯 Problem & Solution

### Problem
- Manual email sending is time-consuming (10+ hours/week)
- Risk of Gmail account suspension from bulk sending
- No way to track who received emails
- Human errors in email addresses and content

### Solution
- **Automated email sending** with rate limiting (20/hour)
- **Excel file upload** for bulk operations
- **Real-time tracking** of email status
- **Professional UI** for easy management

---

## ✨ Key Features

### 1. Smart Email Automation
- Upload Excel file → Emails sent automatically
- Rate limited to 20 emails/hour (Gmail safe)
- 3-minute delay between emails
- Prevents account suspension

### 2. Beautiful User Interface
- Drag & drop file upload
- Real-time statistics dashboard
- Animated gradient design
- Mobile-responsive

### 3. Data Management
- View all students in table
- Track email delivery status
- Delete individual or all records
- Mobile number storage (future SMS)

### 4. Professional Email Templates
- HTML email with gradient design
- Personalized with student name
- Clickable enrollment button
- Plain text fallback
